VoiceForce プライバシーポリシー

Salesforce Voice Assistant Chrome 拡張機能

最終更新日：2026年2月21日  |  バージョン：1.0

VoiceForce（以下「本拡張機能」）は、Salesforce を音声で操作するための Chrome 拡張機能です。 本プライバシーポリシーは、本拡張機能が収集・利用するデータについて説明します。

1. 基本方針

顧客データはサーバーに送信されません。
Salesforce に保存された顧客データ（商談名、取引先名、連絡先情報などのレコード内容）は、 本拡張機能のバックエンドサーバーを一切経由しません。 音声操作の実行は、ユーザーのブラウザから Salesforce REST API に直接通信することで行います。

2. 収集するデータ

本拡張機能が収集するデータは以下のとおりです。

データ種別内容用途保存先
発話テキスト 音声認識により変換されたテキスト（例：「商談一覧を表示して」） LLM による意図解析 Cloudflare Workers（リクエスト処理後に破棄）
Salesforce メタデータ オブジェクト名・項目名の一覧（例：「Opportunity」「Amount」）。
レコードの内容は含まれません。		 意図解析の精度向上 Cloudflare Workers（リクエスト処理後に破棄）
利用回数カウント 匿名のユーザー ID に紐付いた API 呼び出し回数 無料プランの利用制限管理 Cloudflare KV（数値のみ、個人を特定できる情報なし）
OAuth トークン Salesforce アクセストークン・リフレッシュトークン Salesforce API 認証 ユーザーのブラウザ（chrome.storage.local に AES-256-GCM 暗号化して保存）

3. 収集しないデータ

以下のデータは収集・送信しません。

  • Salesforce のレコード内容（商談名・金額・取引先名・連絡先情報など）
  • 氏名・メールアドレス・電話番号などの個人識別情報
  • Salesforce のパスワード・セキュリティトークン
  • ブラウザの閲覧履歴・Cookie・その他のウェブサイトのデータ
  • マイクの音声データ（音声認識はブラウザの Web Speech API により端末内で処理されます）

4. データの利用目的

収集したデータは以下の目的にのみ使用します。

  • 意図解析：発話テキストとメタデータを AI モデル（Claude Haiku / GPT-4o-mini）に送信し、実行すべき Salesforce 操作を特定します。
  • 利用制限管理：無料プランの API 呼び出し上限を管理します（匿名カウントのみ）。
  • サービス改善：将来的に精度向上のため集計データを分析する場合があります（その際は本ポリシーを更新します）。

5. 第三者へのデータ提供

本拡張機能は以下の第三者サービスを利用します。

サービス提供者送信されるデータプライバシーポリシー
Claude API (Haiku) Anthropic, PBC 発話テキスト・メタデータ anthropic.com/privacy
Cloudflare Workers Cloudflare, Inc. 発話テキスト・メタデータ・利用回数 cloudflare.com/privacypolicy
Salesforce REST API Salesforce.com, Inc. OAuth トークン（ブラウザから直接通信） salesforce.com/company/privacy
Web Speech API Google LLC（Chrome 内蔵） 音声データ（端末内処理） policies.google.com/privacy

上記以外の第三者にデータを販売・提供することはありません。

6. データの保存期間

  • 発話テキスト・メタデータ：Cloudflare Workers によるリクエスト処理完了後、直ちに破棄されます。
  • 利用回数カウント：Cloudflare KV に保存され、プラン管理のために保持されます。
  • OAuth トークン：ユーザーが拡張機能をアンインストールするか、手動でログアウトするまでブラウザに保存されます。

7. セキュリティ

  • OAuth アクセストークン・リフレッシュトークンは AES-256-GCM で暗号化して chrome.storage.local に保存します。
  • 暗号化キーは chrome.storage.session（ブラウザセッション中のみ有効）に保存します。
  • Cloudflare Workers との通信は HTTPS で行います。
  • LLM から返されるアクション指示はホワイトリスト検証を経てから実行されます（プロンプトインジェクション対策）。

8. ユーザーの権利

  • データの削除：拡張機能をアンインストールすると、ブラウザに保存されたトークン・設定データはすべて削除されます。
  • ログアウト：拡張機能のポップアップから Salesforce アカウントのログアウトが可能です。
  • 利用回数データの削除：削除をご希望の場合は下記の連絡先までお問い合わせください。

9. 対象ユーザー

本拡張機能は企業での業務利用を想定しており、13歳未満の子どもを対象としていません。 13歳未満の方は本拡張機能を使用しないでください。

10. プライバシーポリシーの変更

本ポリシーを変更する場合は、このページを更新するとともに、 拡張機能の更新履歴（Chrome Web Store）でお知らせします。 重大な変更がある場合は、拡張機能内の通知でもお知らせします。
